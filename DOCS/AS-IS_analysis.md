# AS-IS: Mövcud Layihə İdarəetmə Prosesi

## Alət: Jira (və ya Excel)

## PM-in Gündəlik Fəaliyyəti:
1. Səhər stand-up — 15 dəq
2. Jira-da yeni task yaratmaq — 30 dəq
3. Status izləmə (kim nə edir) — 1 saat
4. Risklərin Excel-də qeydi — 30 dəq
5. Həftəlik status report — 2 saat

## Proses Xəritəsi:
Input (tələb) → Task yaradılır → Owner təyin olunur → 
İcra → Status update → Risk monitorinqi → Report

## AI ilə Avtomatlaşdırıla Bilən Addımlar:
- ✅ Task generasiyası (mətndən)
- ✅ Prioritetləşmə
- ✅ Deadline risk təhlili
- ✅ Status report yazımı
- ❌ Komanda ilə danışıq (insan)
- ❌ Strateji qərarlar (insan)
